# SmartChat

SmartChat is a web-based chat application built with Flask and MySQL.

## Features
- User authentication
- AI-powered chat
- Group chats with admin management
- Direct messages
- Friend system
- Timezone-aware AI responses

## Technologies Used
- Python (Flask)
- MySQL
- SQLAlchemy
- HTML, CSS, JavaScript
- OpenAI API

## Notes
API keys are stored securely using environment variables.
